🚀 Deployment Steps

1

Update Supabase Credentials

Edit js/supabase-config.js with your actual Supabase URL and anon key.

2

Deploy to Netlify

Drag & drop all files to Netlify dashboard, or connect your GitHub repository.

3

Set Environment Variables

In Netlify dashboard, add your Supabase URL and anon key as environment variables.

4

Test Your App

Visit your Netlify URL, register, and start using your trading journal!
